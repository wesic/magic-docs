# Configuration Guide

Learn how to configure and customize PWA Plus to match your brand and business requirements.

## Accessing Configuration Settings

1. Log into your **WHMCS Admin Area**
2. Navigate to **Setup > Addon Modules**
3. Find **PWA Plus** and click **Configure**
4. You'll see the PWA Plus configuration dashboard

## Basic Settings

### App Name & Description

- **Client Area App Name**: e.g., "MyHost Client Portal"
- **Admin Area App Name**: e.g., "MyHost Admin Dashboard"
- **App Description**: Brief description shown during installation (max 200 characters)

### App Icons

Upload custom icons for your PWA:
- **Recommended sizes**: 192x192px and 512x512px
- **Format**: PNG with transparent background
- **Square design** works best for all platforms

### Theme Colors

Customize your app's color scheme:
- **Primary Color**: Main brand color (hex format)
- **Background Color**: App background
- **Theme Color**: Browser toolbar color

## Feature Configuration

### Enable/Disable PWA

- Enable for Client Area
- Enable for Admin Area
- Enable both (recommended)

### Offline Mode

- Enable Offline Support
- Configure cache strategy
- Set cache duration

## Testing Your Configuration

After making changes:
1. Clear browser cache
2. Reload the page (Ctrl+Shift+R or Cmd+Shift+R)
3. Test installation on different devices
4. Verify offline mode
5. Check appearance on mobile and desktop
