# Deployment Guide

Best practices to launch PWA Plus in production.

## Pre-Deployment Checklist

<Card padding="20" radius="l" border="neutral-alpha-medium" fillWidth marginBottom="24">
  <Column gap="12">
    <Text variant="label-strong-m" onBackground="neutral-strong">Technical Requirements</Text>
    <ul style={{margin: 0, paddingLeft: "20px"}}>
      <li>HTTPS enabled with valid SSL certificate</li>
      <li>WHMCS 8.0+ installed and updated</li>
      <li>PHP 7.4 or 8.x configured</li>
      <li>PWA Plus module installed and activated</li>
      <li>Server compatibility verified (cPanel, Plesk, DirectAdmin, or custom)</li>
    </ul>
  </Column>
</Card>

<Card padding="20" radius="l" border="neutral-alpha-medium" fillWidth marginBottom="24">
  <Column gap="12">
    <Text variant="label-strong-m" onBackground="neutral-strong">Configuration Complete</Text>
    <ul style={{margin: 0, paddingLeft: "20px"}}>
      <li>App names set for client and admin</li>
      <li>Icons uploaded (192x192, 512x512 PNG)</li>
      <li>Theme colors customized</li>
      <li>Offline mode configured</li>
      <li>Service worker tested</li>
    </ul>
  </Column>
</Card>

## HTTPS Requirement

<Grid fillWidth gap="16" columns="2" s={{columns: 1}} marginBottom="32">
  <Card padding="16" radius="m" border="neutral-alpha-medium">
    <Column gap="8">
      <Text variant="label-strong-s" onBackground="neutral-strong">Verify HTTPS</Text>
      <Text variant="body-default-s" onBackground="neutral-weak">Padlock in address bar, no mixed-content warnings, valid certificate chain.</Text>
    </Column>
  </Card>
  <Card padding="16" radius="m" border="neutral-alpha-medium">
    <Column gap="8">
      <Text variant="label-strong-s" onBackground="neutral-strong">Force Redirect</Text>
      <Text variant="body-default-s" onBackground="neutral-weak">Enable 301 redirect from HTTP to HTTPS in your web server config.</Text>
    </Column>
  </Card>
</Grid>

## Performance & Caching

<Card padding="20" radius="l" border="neutral-alpha-medium" fillWidth marginBottom="24">
  <Column gap="10">
    <Text variant="label-strong-m" onBackground="neutral-strong">Recommended Server Settings</Text>
    <Card padding="12" radius="m" background="neutral-alpha-weak">
      <Text variant="label-default-s" onBackground="neutral-medium" style={{fontFamily: "monospace", whiteSpace: "pre"}}>
memory_limit = 256M
max_execution_time = 300
upload_max_filesize = 64M
post_max_size = 64M
      </Text>
    </Card>
    <Text variant="body-default-s" onBackground="neutral-weak">Ensure appropriate permissions: 755 for directories, 644 for files in `/modules/addons/pwaplus`.</Text>
  </Column>
</Card>

<Grid fillWidth gap="16" columns="2" s={{columns: 1}} marginBottom="32">
  <Card padding="16" radius="m" border="neutral-alpha-medium">
    <Column gap="8">
      <Text variant="label-strong-s" onBackground="neutral-strong">Caching</Text>
      <Text variant="body-default-s" onBackground="neutral-weak">Use cache-first for static assets; network-first for APIs; long TTL for images and fonts.</Text>
    </Column>
  </Card>
  <Card padding="16" radius="m" border="neutral-alpha-medium">
    <Column gap="8">
      <Text variant="label-strong-s" onBackground="neutral-strong">Audit</Text>
      <Text variant="body-default-s" onBackground="neutral-weak">Run Lighthouse PWA audit; aim for 90+ on Performance, Best Practices, PWA.</Text>
    </Column>
  </Card>
</Grid>

## Launch Checklist

<Card padding="20" radius="l" border="neutral-alpha-medium" fillWidth marginBottom="24">
  <Column gap="8">
    <Text variant="label-strong-m" onBackground="neutral-strong">Launch Checklist</Text>
    <ul style={{margin: 0, paddingLeft: "20px"}}>
      <li>Final testing in staging</li>
      <li>Backup WHMCS before release</li>
      <li>Deploy PWA Plus to production</li>
      <li>Verify install prompt on multiple devices</li>
      <li>Monitor errors and performance for first 24h</li>
    </ul>
  </Column>
</Card>

## Post-Launch Monitoring

- Installation success rate
- Service worker status and errors
- Page load performance (Core Web Vitals)
- User engagement (installs, active users)
