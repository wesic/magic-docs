## Installation Steps

### Step 1: Download PWA Plus Module

<Card padding="20" radius="m" border="neutral-alpha-medium" fillWidth marginBottom="20">
  <Column gap="12">
    <Text variant="label-strong-m" onBackground="neutral-strong">Download the Module</Text>
    <ol style={{paddingLeft: "20px", margin: 0}}>
      <li>Download the PWA Plus module package from your purchase source</li>
      <li>Extract the downloaded ZIP file on your local computer</li>
      <li>You should see a folder named `pwaplus`</li>
    </ol>
  </Column>
</Card>

### Step 2: Upload Module Files

<Card padding="20" radius="m" border="neutral-alpha-medium" fillWidth marginBottom="20">
  <Column gap="16">
    <Text variant="label-strong-m" onBackground="neutral-strong">Upload to WHMCS</Text>
    
    <Column gap="12">
      <Text variant="label-default-s" onBackground="neutral-medium">Via FTP/SFTP:</Text>
      <ol style={{paddingLeft: "20px", margin: 0}}>
        <li>Connect to your server using FTP/SFTP client (FileZilla, WinSCP, etc.)</li>
        <li>Navigate to your WHMCS installation directory</li>
        <li>Go to `/modules/addons/`</li>
        <li>Upload the entire `pwaplus` folder to this directory</li>
      </ol>
    </Column>

    <Column gap="12">
      <Text variant="label-default-s" onBackground="neutral-medium">Via cPanel File Manager:</Text>
      <ol style={{paddingLeft: "20px", margin: 0}}>
        <li>Log into your cPanel</li>
        <li>Open File Manager</li>
        <li>Navigate to your WHMCS directory</li>
        <li>Go to `/modules/addons/`</li>
        <li>Upload and extract the module ZIP file here</li>
      </ol>
    </Column>

    <Card padding="12" radius="s" background="brand-alpha-weak" marginTop="8">
      <Text variant="label-default-s" onBackground="brand-medium">
        **Final path should be:** `/path/to/whmcs/modules/addons/pwaplus/`
      </Text>
    </Card>
  </Column>
</Card>

### Step 3: Activate the Module

<Card padding="20" radius="m" border="neutral-alpha-medium" fillWidth marginBottom="20">
  <Column gap="12">
    <Text variant="label-strong-m" onBackground="neutral-strong">Activate in WHMCS</Text>
    <ol style={{paddingLeft: "20px", margin: 0}}>
      <li>Log into your **WHMCS Admin Area**</li>
      <li>Navigate to **Setup > Addon Modules**</li>
      <li>Find **PWA Plus** in the list of available modules</li>
      <li>Click the **Activate** button</li>
      <li>Configure access control if needed (grant access to admin roles)</li>
    </ol>
  </Column>
</Card>

### Step 4: Access Module Configuration

<Card padding="20" radius="m" border="neutral-alpha-medium" fillWidth marginBottom="32">
  <Column gap="12">
    <Text variant="label-strong-m" onBackground="neutral-strong">Initial Configuration</Text>
    <ol style={{paddingLeft: "20px", margin: 0}}>
      <li>After activation, click **Configure** next to PWA Plus</li>
      <li>You'll be taken to the PWA Plus settings page</li>
      <li>Configure your app settings (see [Configuration Guide](/pwa-plus/setup-guide/configuration))</li>
    </ol>
  </Column>
</Card>
