# Frequently Asked Questions

## General Questions

<Accordion title="What is PWA Plus?">
PWA Plus is a WHMCS addon module that transforms your Client Area and Admin Area into installable Progressive Web Apps that work like native mobile and desktop applications.

**Key benefits include:**
- Installable on all major platforms
- Works offline with smart caching
- No app store submissions required
- Automatic updates
- Native app-like experience
</Accordion>

<Accordion title="Does PWA Plus modify WHMCS core files?">
No, PWA Plus is a **completely standalone addon**. It does not modify any WHMCS core files and follows official WHMCS development standards. This makes it safe to install and easy to update without affecting your WHMCS installation.
</Accordion>

<Accordion title="Do I need to publish the app to app stores?">
No! The PWA is installable directly from your browser without any app store submissions, approvals, or fees. Users simply visit your WHMCS and can install the app with one click directly from their browser.
</Accordion>

<Accordion title="What devices and platforms are supported?">
PWA Plus works on:
- **Mobile**: Android, iOS/iPadOS
- **Desktop**: Windows, macOS, Linux
- **Browsers**: Chrome, Edge, Safari, Firefox, and other modern browsers

The app provides a consistent experience across all platforms.
</Accordion>

## Installation & Setup

<Accordion title="What are the requirements for PWA Plus?">
**System Requirements:**
- WHMCS 8.0 or later
- PHP 7.4 or PHP 8.x
- HTTPS (SSL certificate required)
- Compatible with cPanel, Plesk, DirectAdmin, and custom servers

All modern hosting environments support these requirements.
</Accordion>

<Accordion title="Why is HTTPS required?">
Progressive Web Apps require HTTPS for security reasons. This is a standard requirement for all PWAs, not just PWA Plus. HTTPS ensures that:
- Data transmission is encrypted
- User information is protected
- Service workers can function properly
- The app can be trusted by browsers

Most hosting providers offer free SSL certificates via Let's Encrypt.
</Accordion>

<Accordion title="How long does installation take?">
Installation typically takes **5-10 minutes**. Simply:
1. Upload the module files
2. Activate it in WHMCS
3. Configure your preferences
4. Test the installation

Our [Installation Guide](/pwa-plus/setup-guide/installation) provides step-by-step instructions.
</Accordion>

## Features & Functionality

<Accordion title="Can I customize the app's appearance?">
Yes! You can customize:
- App name and icon
- Brand colors and theme
- UI elements
- Overall look and feel

The configuration is done through the WHMCS admin panel without any coding required. See our [Configuration Guide](/pwa-plus/setup-guide/configuration) for details.
</Accordion>

<Accordion title="Does it work offline?">
Yes, PWA Plus includes offline mode support. Critical pages and assets are cached, allowing users to access key features even without internet connectivity. When the connection is restored, the app automatically syncs any changes.

**Available offline:**
- View recent invoices
- Read support tickets
- Browse service information
- Access account details
</Accordion>

<Accordion title="Are push notifications available?">
Push notifications are **currently under development** and will be released in an upcoming update. Once available, you'll be able to send alerts for:
- Invoices and payment reminders
- Support ticket updates
- Service renewals
- Custom notifications

We'll announce the release through our changelog and support channels.
</Accordion>

<Accordion title="Can I have separate apps for clients and admins?">
Yes! PWA Plus creates separate installable PWAs for both the Client Area and Admin Dashboard. Each has its own icon, branding, and can be configured independently to serve different user needs.
</Accordion>

## Performance & Maintenance

<Accordion title="Will PWA Plus slow down my WHMCS?">
No, PWA Plus actually **improves performance** through:
- Smart caching of assets
- Optimized asset loading
- Reduced server requests
- Instant page loads from cache

Users will experience faster load times compared to traditional web browsing.
</Accordion>

<Accordion title="Do I need to manually update the app?">
No maintenance required! When you deploy updates:
- Users' apps automatically update in the background
- No app store submissions needed
- No manual client updates required
- Updates happen seamlessly

Users always have the latest version without any action on their part.
</Accordion>

<Accordion title="Does it require any external services or subscriptions?">
No. PWA Plus is completely self-contained with:
- **No third-party dependencies**
- **No monthly fees**
- **No external API requirements**
- **No data sharing with external services**

Everything runs on your server with full data privacy.
</Accordion>

## Compatibility

<Accordion title="Is PWA Plus compatible with my WHMCS theme?">
PWA Plus works independently of your WHMCS theme and provides its own optimized mobile-first interface. This ensures a consistent, high-quality experience regardless of your main WHMCS theme.
</Accordion>

<Accordion title="Does it work with my server control panel?">
Yes, PWA Plus is compatible with:
- cPanel
- Plesk
- DirectAdmin
- Custom server configurations

As long as you meet the PHP and HTTPS requirements, PWA Plus will work on your server.
</Accordion>

<Accordion title="Can I use it with other WHMCS modules?">
Yes, PWA Plus is designed to work alongside other WHMCS modules without conflicts. It follows WHMCS development best practices and doesn't interfere with other addons or integrations.
</Accordion>
