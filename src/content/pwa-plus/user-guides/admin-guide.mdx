## Module Management

- **Access settings:** Setup → Addon Modules → PWA Plus → Configure
- **Toggle availability:** Enable/disable PWA for Client and Admin areas
- **Branding:** Update app name, description, icons, theme colors
- **Caching:** Adjust cache strategy and duration
- **Access control:** Restrict configuration to specific admin roles

## Troubleshooting for Admins

- **Module missing:** Verify upload to `/modules/addons/pwaplus/`, permissions 755/644, PHP 7.4/8.x
- **Cannot configure:** Check admin role permissions; clear browser cache; ensure HTTPS
- **PWA not installing:** Confirm HTTPS, manifest accessible, service worker registered

## Best Practices

- Keep WHMCS and PWA Plus updated
- Use strong SSL and enforce HTTPS
- Test on multiple devices before releases
- Promote install banners to clients
