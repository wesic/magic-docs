## What to Include in Your Ticket

<Card padding="24" radius="l" border="neutral-alpha-medium" marginBottom="32">
  <Column gap="16">
    <Text variant="heading-strong-m" onBackground="neutral-strong">For faster resolution, please provide:</Text>
    
    <Column gap="12">
      <Text variant="heading-strong-s" onBackground="neutral-strong">System Information:</Text>
      <Column gap="6">
        <Text variant="body-default-s" onBackground="neutral-weak">• WHMCS version</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• PHP version</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Server type (cPanel, Plesk, etc.)</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• PhonePe Payment Gateway version</Text>
      </Column>
    </Column>
    
    <Column gap="12">
      <Text variant="heading-strong-s" onBackground="neutral-strong">Issue Details:</Text>
      <Column gap="6">
        <Text variant="body-default-s" onBackground="neutral-weak">• Clear description of the issue</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Steps to reproduce</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Expected vs actual behavior</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Transaction IDs (if payment-related)</Text>
      </Column>
    </Column>
    
    <Column gap="12">
      <Text variant="heading-strong-s" onBackground="neutral-strong">Supporting Materials:</Text>
      <Column gap="6">
        <Text variant="body-default-s" onBackground="neutral-weak">• Screenshots or screen recordings</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Error messages (full text)</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Browser console logs</Text>
        <Text variant="body-default-s" onBackground="neutral-weak">• Server error logs</Text>
      </Column>
    </Column>
  </Column>
</Card>
